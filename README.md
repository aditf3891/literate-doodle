# Aplikasi Generate Barcode

Aplikasi web sederhana untuk membuat barcode langsung dari browser.

## Fitur
- Generate barcode dari teks/angka.
- Pilihan format: CODE128, EAN13, UPC, ITF14, MSI.
- Pengaturan lebar garis dan tinggi barcode.
- Download hasil sebagai PNG.

## Menjalankan
Cukup buka `index.html` di browser, atau jalankan server lokal:

```bash
python3 -m http.server 8000
```

Lalu buka `http://localhost:8000`.
